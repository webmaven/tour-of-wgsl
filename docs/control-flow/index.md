---
title: 'Control Flow'
---

WGSL provides structured control flow statements for branching and looping:

- **[If Statements](if-statements.md)**: Conditional branching using `if` and `else`.
- **[Switch Statements](switch-statements.md)**: Multi-way branching based on discrete scalar values.
- **[Loop Statements](loop-statements.md)**: Infinite loops with `loop` and control features like `continuing`.
- **[While Statements](while-statements.md)**: Standard pre-checked loops with `while`.
- **[For Statements](for-statements.md)**: Iterative loops with initialization, condition, and update expressions.
