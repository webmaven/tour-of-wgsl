---
title: 'Types'
---

WGSL supports a wide range of types to enable efficient GPU programming:

- **[Basic Scalars](basic-scalars.md)**: Learn about standard scalar types like `bool`, `i32`, `u32`, and `f32`.
- **[Abstract Numerics](abstract-numerics.md)**: Explore compile-time abstract integer (`abstract-int`) and abstract float (`abstract-float`) types.
- **[Vectors](vectors/index.md)**: Work with multi-component vectors (`vec2`, `vec3`, `vec4`) and component swizzling.
- **[Matrices](matrices/index.md)**: Explore column-major matrices (e.g., `mat4x4<f32>`) and matrix math.
- **[Arrays](arrays/index.md)**: Declare fixed-size and runtime-sized arrays.
- **[Structures](structures/index.md)**: Define custom user types with specific memory alignments.
- **[Pointers](pointers/index.md)**: Understand address spaces, pointer syntax, and strict aliasing rules.
- **[Atomics](atomics/index.md)**: Coordinate memory access across invocations using lock-free atomic types.
